Your website, now with a voice

This page is plain HTML. The floating button in the bottom-right corner is the entire integration — added with a single <script> tag. Click it and talk to an Omni voice agent, right in the browser. No phone, no Twilio.

To add the same widget to your own site, paste this one line:

<script src="https://your-site.example/pyai-widget.js" data-token-url="/token"></script>

data-token-url points at your backend, which mints a short-lived session token so your secret key never ships to the browser (see server.js). Your key never touches this page.
